This website displays the images stored in an Azure Storage account. It is a custom built image gallery that provides unlimited storage to host your memorable pictures.

  • Supported Video Formats: 'MP4','MOV','WMV','FLV','AVI','AVCHD','WEBM','MKV'
  • Supported Image Formats: 'JPEG','JPG','GIF','PNG','TIFF','TIF','BMP','EPS','RAW','CR2','NEF','ORF','SR2'
Note that mentioned formats are case insensitive. Any other file apart from these supported formats is ignored by the application, however it will still persist in the backend storage.
